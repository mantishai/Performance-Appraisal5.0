# 人力资源管理系统 (HRMS)

## 项目介绍

企业级人力资源管理系统，覆盖人事管理全流程。

## 功能模块

- 13个核心模块 + 7个增强模块

## 技术栈

- 前端：HTML5 + CSS3 + JavaScript ES6+
- 后端：Node.js + Express + MySQL
- 数据库：MySQL 8.0
- 缓存：Redis 7.0
- 部署：Docker + Nginx

## 快速开始

### 环境要求

- Node.js >= 16
- MySQL >= 8.0
- Docker（可选）

### 安装

```bash
git clone https://github.com/mantishai/Performance-Appraisal5.0
cd hrms
npm install
```

### 开发

```bash
npm run dev
```

### 构建

```bash
npm run build
```

### 部署

```bash
npm run deploy
```
